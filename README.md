# PhotoCatch - 图片采集工具

一个基于Python的图片采集工具，提供图形界面，可以根据关键字从指定网站爬取图片并保存到本地。

## 功能特点

- **图形界面**：使用Python内置tkinter库，无需额外安装GUI框架
- **多线程爬取**：后台线程执行爬取任务，界面保持响应
- **智能解析**：自动分析网页源代码，提取图片URL
- **用户模拟**：设置真实的User-Agent，模仿浏览器访问
- **进度显示**：实时显示爬取进度和日志信息
- **可配置**：支持自定义请求延迟、User-Agent等参数

## 环境要求

- Python 3.10+ (建议使用miniconda管理虚拟环境)
- pip 包管理器

## 安装步骤

### 1. 创建虚拟环境（推荐使用miniconda）

```bash
# 使用miniconda创建虚拟环境
conda create -n photocatch python=3.10

# 激活虚拟环境
conda activate photocatch
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包说明：

- `requests`：HTTP请求库，用于发送网络请求
- `beautifulsoup4`：HTML解析库，用于分析网页结构
- `lxml`：XML/HTML解析器，提高解析效率
- `Pillow`：图像处理库，用于验证和保存图片

## 使用方法

### 图形界面启动

```bash
python main.py
```

### 界面操作说明

1. **目标网站**：输入要爬取的网站URL（默认为百度图片）
2. **关键词**：输入搜索关键词，多个关键词用逗号分隔（用于过滤相关页面）
3. **图片数量**：设置要下载的图片数量（1-1000）
4. **保存目录**：选择图片保存的本地文件夹
5. **请求延迟**：设置每次请求的间隔时间（秒），建议不要小于0.5秒，避免被封IP
6. **开始采集**：点击开始按钮启动爬取任务
7. **停止采集**：点击停止按钮中断爬取任务

## 工作原理

1. **页面访问**：使用requests库发送HTTP请求，设置真实的User-Agent和请求头
2. **HTML解析**：使用BeautifulSoup分析网页源代码，提取<img>标签
3. **图片识别**：从<img>标签的src、data-src、data-lazy等属性中提取图片URL
4. **链接跟随**：根据关键词过滤页面中的链接，深度爬取相关页面
5. **图片下载**：下载图片并使用Pillow验证图片格式，保存到本地

## 注意事项

1. **法律合规**：请确保您有权爬取目标网站的内容，遵守网站的robots.txt规则
2. **请求频率**：建议设置合理的请求延迟（1秒以上），避免对目标服务器造成压力
3. **内容版权**：下载的图片可能受版权保护，请确保使用合法
4. **反爬虫机制**：部分网站可能有反爬虫机制，可能导致爬取失败

## 故障排查

### 问题1：无法导入模块

**解决方法**：

```bash
# 确保已安装所有依赖
pip install -r requirements.txt

# 检查Python版本
python --version
```

### 问题2：爬取失败，返回403错误

**解决方法**：

1. 检查目标网站是否允许爬取
2. 增加请求延迟时间
3. 修改User-Agent字符串

### 问题3：GUI界面无响应

**解决方法**：

1. 爬取任务在后台线程执行，不会阻塞界面
2. 如果界面长时间无响应，尝试重启程序
